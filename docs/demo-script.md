# Three-minute demo

1. Start the stack and open the dashboard and Jaeger.
2. Run **Healthy** to establish a green baseline.
3. Run **Slow dependency**. Show recommendation latency, retries, checkout p99, and linked evidence.
4. Compare the three GPT-5.6 fixture candidates and their validation results.
5. Run counterfactual simulation and then bounded live replay for the safest candidate.
6. Approve the candidate. Show the TTL and the typed active policy.
7. Replay traffic and show checkout recovery with zero rejected critical requests.
8. Deactivate or wait for expiry and explain that no raw model output reached the service.

# Prior work versus build-week work

SLO Guardian is inspired by prior exploration of trace-aware load shedding. This repository starts
as a clean implementation. Build-week work includes the five-service demo, deterministic scenario
engine, OpenTelemetry pipeline, SLO calculations, GPT-5.6 structured reasoning boundary, evidence
validation, policy safety system, replay simulator, operator approval flow, and React dashboard.

If code is imported later, record its source commit, license, original author, and the exact changes
made during the hackathon in this document.
